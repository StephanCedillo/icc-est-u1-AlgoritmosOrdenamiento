## Diferencias entre ambos algoritmos:
Insertion Sort es un algoritmo de ordenamiento simple que construye la lista ordenada uno a uno, insertando cada elemento en su posición correcta. Es eficiente para listas pequeñas o casi ordenadas, ya que va comparando cada elemento con los anteriores y moviéndolos si es necesario cumple la condicion.
Shell Sort es una variante del Insertion Sort que divide la lista en subsecuencias más pequeñas y las ordena de forma independiente mediante Insert Sort , lo que mejora su eficiencia en comparación con el Insertion Sort tradicional.

## Conclusiones:
Ambos algoritmos de ordenamiento tienen sus propias ventajas y desventajas. Insertion Sort es fácil de entender e implementar Shell Sort, por otro lado, es más eficiente ya que divide la lista a la mitad como el algoritmo log(n). Es importante elegir el algoritmo de ordenamiento adecuado según el tamaño y los datos que se están ordenando.
